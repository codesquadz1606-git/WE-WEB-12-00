What is diffrence b/w let var & const?

let keyword says that it cannot be redeclared but we can re-assign the value.

var keyword says that it can be redeclared as well as re-assign.

const keyword cannot be redeclared as well as re-assign.

What is Data-Type?
Data-Type means which type of value is stored in a variable.

--> Number : any int & float(decimal) number
--> String : any character inside "".
--> Boolean : true or false
--> undefined : where variable has no value.
--> null
