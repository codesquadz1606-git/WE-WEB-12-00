String: String is a set of characters store in a "" is known as string.

String is based on zero base indexing,
STARTS FROM ZERO & GOES TILL LENGTH-1.

String function

    lastIndexOf()
    includes()

    endsWith()
    search()
2. Extraction
    slice(start,end-1)
    charAt()
    at()
3. Modification
    replace()
    replaceAll()
    toUpperCase()
    toLowerCase()
    trim(),trimStart(),trimEnd() : removes the extra spaces.
    repeat()
    concat()
4. Splting & Joining
    split()
    join()
